# Process Manager Simulator

An Operating Systems course project that simulates CPU scheduling algorithms
including FCFS, SJF, and Round Robin with an ML-based dynamic time quantum.

## Tech Stack
- Python
- Streamlit
- NumPy
- Pandas
- Scikit-learn
- Joblib

## How to Run
python3 -m pip install -r requirements.txt  
python3 -m streamlit run app.py
